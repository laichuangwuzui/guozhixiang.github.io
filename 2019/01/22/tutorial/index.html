Hexo入门教程

2019-01-22

花费了很长的一段时间才将Hexo博客搭建到github上，中间出现了很多的坑，下面的搭建过程希望可以帮助到后面进行学习的人。

Hexo是一款“快速、简洁且高效的博客框架”，那些尝试搭建个人网站的同学，或者是学习git的小伙伴都可以进行尝试学习。

一、环境前提（不详细赘述）

1.安装Node.js和配置好Node.js环境
2.安装Git和配置好Git环境，安装成功的象征就是在电脑上任何位置鼠标右键能够出现如下两个选择Git GUI Here和Git Bash Here
3.Github账户注册和新建项目，项目必须要遵守格式：账户名.github.io

二、安装Hexo

1.在本地自己创建自己的博客文件夹
下面是我自己创建的文件夹形式


2.打开项目文件夹右键Git Bash Here
执行命令输入npm install hexo -g，开始安装Hexo
输入hexo -v，检查hexo是否安装成功
输入hexo init，初始化该文件夹（有点漫长的等待。。。）
输入npm install，安装所需要的组件
输入hexo g，首次体验Hexo（该过程是编译代码的过程）
输入hexo s，开启服务器，访问该网址，正式体验Hexo


本地服务器地址localhost:4000

到这一步你的本地博客项目已经生成了

三、同步项目到GitHub
1.注意我们的GitHub的仓库名的书写方式 账户名.github.io


2.打开我们的项目下yml文件


修改文件最下面的配置项目


注意yml文件的书写规范‘:’后面要带空格
这三行你需要填写的内容分别为，使用的项目管理工具git,你的github的远程仓库地址，
以及master分支
3.保存文件并部署
部署前我们需要安装一个工具
npm install hexo-deployer-git –save
如果你使用这个命令报错如下


你可以使用cnpm install hexo-deployer-git –save
使用淘宝镜像网站进行插件的安装
安装成功后使用命令
hexo g,进行代码生成
hexo d，进行项目部署
这个时候我们就可以通过远程仓库上的github提供的网址进行访问了


如果通过该网址可以进行访问到项目主页，那么说名你的本地项目已经成功的部署到了github上了
